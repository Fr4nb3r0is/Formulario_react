# Formulario React - Gestión de Personas

Este proyecto es una aplicación sencilla en React para gestionar un listado de personas. Permite agregar registros mediante un formulario y visualizarlos en una tabla.

## Características

- Navbar superior con el nombre de la aplicación
- Formulario para ingresar nombre, apellido y email
- Botón para agregar personas al listado
- Tabla que muestra los registros ingresados
- Sin estilos CSS, solo HTML básico

## Estructura principal

- `src/App.jsx`: Componente principal, gestiona el estado global y renderiza los componentes
- `src/Componente/formulario.jsx`: Contiene los componentes Navbar, Formulario y Tabla
- `src/App.css`: Estilos mínimos para visibilidad del texto

## Cómo ejecutar

1. Instala las dependencias:
   ```bash
   npm install
   ```
2. Ejecuta la aplicación:
   ```bash
   npm run dev
   ```
3. Abre tu navegador en `http://localhost:5173` (o el puerto que indique la terminal)

## Uso

- Completa el formulario y haz clic en "Agregar Persona"
- Los datos aparecerán en la tabla debajo
